
# EPG服務說明
## 一、概述
本電子節目指南（EPG）服務僅面向個人娛樂使用，嚴禁用於商業用途。服務覆蓋大陸、香港、澳門、台灣、韓國、新加坡、馬來西亞、印度尼西亞等地區的電視台節目預告，針對性推出簡體中文與繁體中文兩個版本，适配不同區域用戶需求。

## 二、使用規範
### （一）使用限制
1. 僅限個人非商業用途，任何商業性質的開發、傳播或獲利行為均嚴格禁止；
2. 本服務不承諾提供常規維護及技術支援服務。

### （二）內容與資源說明
1. 不提供節目源地址，用戶需通過訂閱官方正版渠道獲取合法節目資源；
2. YouTube相關鏈接僅提供原始地址，不包含任何解析、解碼服務，建議使用「影視」應用播放（備註：影視直播對XML格式相容性較差）；
3. 推薦播放應用：Ok影視、Tivimate、OTT Navigator、極致播放器。

## 三、EPG服務規格
### （一）推薦格式
優先選擇**xml.gz格式**，相比XML格式，具有文件體積更小、加載速度更快的優勢（原文「xm.gz」為筆誤，已修正）。

### （二）文件架構
1. file.json 用於複製部分EPG文件；
2. kai.php、kai1.php 用於生成HOY電視台EPG；
3. egziyong.php 用於生成綜合性EPG；
4. epggen2.php 用於替換頻道ID（channel id）；
5. epgnewshanghai.php 用於提取上海地方台；
6. epgnewhebei.php 用於提取河北地方台；
7. twepg.xml 繁體版本EPG，適合港澳台地區；
8. swepg.xml 簡體版本EPG，適合內地用戶。

### （三）區域适配版本
| EPG版本 | 文件名 | 適用區域 | 下載鏈接 |
| ---- | ---- | ---- | ---- |
| 簡體中文 | swepg.xml.gz | 大陸、新加坡、馬來西亞 | https://github.com/zzq12345/epgtest/raw/refs/heads/main/swepg.xml.gz |
| 繁體中文 | twepg.xml.gz | 香港、澳門、台灣 | https://github.com/zzq12345/epgtest/raw/refs/heads/main/twepg.xml.gz |

### （四）大陸用戶特別說明
1. 建議選用**epgnew.xml.gz**文件，該版本已剔除少量未滿十八歲不宜及區域限制類電視台的節目預告；
2. 境外用戶可根據需求自由選擇epgziyong.xml或epgnew.xml。

# English Version: EPG Service Instructions
## 1. Service Overview
This EPG (Electronic Program Guide) service is provided exclusively for personal entertainment and non-commercial use. It covers program previews for TV channels across Mainland China, Hong Kong, Macau, Taiwan, South Korea, Singapore, Malaysia, and Indonesia, with dedicated Simplified and Traditional Chinese versions tailored to different regional needs.

## 2. Key Terms & Conditions
### 2.1 Usage Restrictions
1. This service is for **personal non-commercial use only**; commercial exploitation is strictly prohibited.
2. Routine maintenance and technical support are not guaranteed.

### 2.2 Content & Resource Notes
1. Program source URLs are not provided. Users must subscribe to official and legitimate program source services.
2. YouTube links are shared in their original form without any parsing or decoding. Yingshi is recommended for use, though Yingshi Live has poor compatibility with XML formats.
3. Recommended playback applications: Ok Video, Tivimate, OTT Navigator, Jizhi Player.

## 3. EPG Specifications
### 3.1 Recommended Format
**xml.gz format** is preferred over XML, as it offers smaller file size and faster loading speed.

### 3.2 File Structure
1. file.json: For copying some EPG files.
2. kai.php, kai1.php: For generating HOY TV EPG.
3. egziyong.php: For generating comprehensive EPG.
4. epggen2.php: For replacing channel IDs.
5. epgnewshanghai.php: For extracting Shanghai local TV channels.
6. epgnewhebei.php: For extracting Hebei local TV channels.
7. twepg.xml: Traditional Chinese version EPG, suitable for Hong Kong, Macau, and Taiwan.
8. swepg.xml: Simplified Chinese version EPG, suitable for Mainland China users.

### 3.3 Regional Adaptations
| EPG Version | File Name | Applicable Regions | Download URL |
|-------------|-----------|--------------------|--------------|
| Simplified Chinese | swepg.xml.gz | Mainland China, Singapore, Malaysia | https://github.com/zzq12345/epgtest/raw/refs/heads/main/swepg.xml.gz |
| Traditional Chinese | twepg.xml.gz | Hong Kong, Macau, Taiwan | https://github.com/zzq12345/epgtest/raw/refs/heads/main/twepg.xml.gz |

### 3.4 Special Notes for Mainland China Users
1. It is recommended to use **epgnew.xml** (excludes program previews for a few adult-oriented and regionally restricted TV channels).
2. Users outside Mainland China may choose either epgziyong.xml or epgnew.xml.
